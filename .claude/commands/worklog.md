# /log — 작업 로그 기록

현재 대화에서 수행한 작업을 `/Users/jeongeun/Desktop/aaa/00_주차별미션/Week_04/Week_04_비비안.md` 파일에 append하세요.

## 기록 포맷

```markdown
### {작업 제목 — 간결하게}
**왜:** {이 작업을 한 이유 — 어떤 문제를 해결하려 했는지}
**한 일:**
- {구체적으로 한 것 1}
- {구체적으로 한 것 2}
- ...
```

## 규칙

1. 파일 맨 끝에 append (기존 내용 유지)
2. 날짜 헤딩(`## YYYY-MM-DD`)이 오늘 날짜로 이미 있으면 그 아래에 추가. 없으면 새 날짜 헤딩을 먼저 추가.
3. 시간은 기록하지 않음
4. "왜"는 기술적 이유가 아니라 **프로젝트 관점의 동기**를 적을 것
5. "한 일"은 변경한 파일 나열이 아니라 **무엇이 달라졌는지** 중심으로
6. 파일 최상단에 `# Week 04 — 셀피쉬클럽 v4.0 작업 로그` 헤딩이 없으면 추가
