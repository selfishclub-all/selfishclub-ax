셀피쉬클럽 공유회란?

셀피쉬클럽은 이기적멤버십이라는 유료 커뮤니티를 운영한다. 공유회는 외부 전문가를 초청해서 멤버들에게 제공하는 라이브 세미나.
매달 2~3건씩 열리는데, 매번 모집 오픈 → 리마인드 → 마감 → 당일 안내 → 사후 VOD 발송까지 6개 채널에서 수십 개 메시지를 보내야 한다.
이걸 1인 마케터(에밀리)가 혼자 수작업으로 해왔다.

Claude Code = 두뇌

카피 생성, 배너 디자인, UTM 규칙 적용, Slack 전달까지. AI 에이전트 7종이 팀으로 협업. 맥북이 켜져 있을 때 "만드는 작업"을 처리.

n8n = 알람시계

알림톡 9종의 시간별 자동 발송. 크론으로 돌아가서 맥북 꺼도 됨. Slack 모달로 에밀리가 카피 확인/수정 후 승인.

왜 이 구조?

알림톡은 정해진 시간에 정확히 나가야 해서 n8n 크론. 카피/배너는 AI가 잘하는 영역이라 Claude Code. 각각의 강점을 살린 하이브리드.

배너 3종 자동 생성

오픈/리마인드/마감 Phase별 800x600 와이드 배너. Gemini API로 주제 맞는 배경 이미지 생성 → HTML 조합 → PNG 변환. 카피+이미지+UTM 한 세트.

발송 방식

Claude Code가 카피+배너 생성 → Slack에 미리보기 → 에밀리 확인 → SOLAPI 친구톡 예약 발송. 카카오 파트너센터 API가 없어서 SOLAPI 경유.

3방 x 3Phase = 9개 카피

셀피쉬 대화방/AI 대화방/사이드프로젝트 대화방. 각 방마다 다른 톤의 카피 + 각각 다른 bit.ly (UTM 추적용). 오픈/리마인드/마감 3Phase.

발송 방식

Claude Code가 카피 자동 생성 → Slack #crm-카피에 복붙용으로 전달 → 에밀리가 카카오톡 오픈채팅방에 직접 복붙. (오카방 API 없음)

온드미디어 4종

리틀리 (랜딩페이지) / 대화방소식 / 웰컴메시지 / 소식 — 각각 카피 + UTM URL 자동 생성. CRM 시작 시 한번에 생성되어 Slack 스레드로 전달.

발송 방식

에밀리가 Slack 카피를 각 플랫폼에 수동 세팅. 저빈도(공유회 당 1회)라서 수동이 합리적. 공유회 1건당 약 15분 소요.

캐러셀 5장 자동 생성

1080x1080 인스타그램 캐러셀 5장. 프로그램 정보 기반 카피 + Gemini 배경이미지 + HTML 조합 → PNG. 캡션 + 해시태그도 자동.

발송 방식

현재: Slack에 이미지 미리보기 전달 (게시는 수동). 추후: Meta Graph API로 자동 게시 예정.

기술 스택

실제 사용된 도구와 선택 이유

🤖

Claude Code (Opus/Sonnet/Haiku)

AI 에이전트 팀 오케스트레이션. 카피 생성, 코드 실행, API 호출까지 CLI 하나로.

⚙

n8n (Self-hosted)

알림톡 크론 자동화. Slack Block Kit 모달. 맥북 꺼도 돌아가는 독립 워크플로우.

🗃

Supabase (PostgreSQL)

회원 DB, 프로그램 DB, 신청 내역. n8n과 Claude Code 모두 접근.

💬

SOLAPI

알림톡 + 친구톡 발송 API. 카카오 비즈니스 채널 연동. 템플릿 변수 치환.

💬

Slack MCP

카피 미리보기 전달, 승인 인터페이스, 팀 커뮤니케이션 허브.

🎨

Gemini API

카플친 배너/인스타 캐러셀 배경 이미지 생성. 텍스트 없이 주제만.

🔗

Bitly API

오카방 3방 각각 다른 단축URL (UTM 추적용). 21개 UTM 규칙 자동 적용.

📄

PRD 9개 + 에이전트 6종

삼단구조: 스킬(파이프라인) + 에이전트(역할) + PRD(규칙 원본). 2,500줄+ 문서.
