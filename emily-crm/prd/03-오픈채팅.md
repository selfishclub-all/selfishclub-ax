# 03 — 오픈채팅 (카카오 오카방) PRD

## 개요
- 카카오 오픈채팅 3개 방에 Phase별 카피 전달
- API 미지원 → Claude 카피 생성 → Slack 전달 → 에밀리 수동 게시
- **3방 동일 카피** × 3Phase = 3개 카피 (bit.ly URL만 방별로 다름)

## 3개 방
| 방 | bit.ly UTM content 구분 |
|---|------------------------|
| 셀피쉬냅다활용 | `{YYMMDD}_{Phase}_냅다활용` |
| 셀피쉬냅다실무 | `{YYMMDD}_{Phase}_냅다실무` |
| 셀피쉬클럽 | `{YYMMDD}_{Phase}_클럽` |

**카피 내용은 3방 완전 동일.** bit.ly URL만 UTM content가 달라서 어느 방에서 유입됐는지 추적.

## 카피 톤 규칙
- **자연스러운 구어체**: "~거든요", "~더라구요", "~이에요", "~해보세요"
- **AI 티 절대 금지**: "~했습니다. ~하고요." 식 setup→punchline 금지
- **비개발자 타겟일 때**: 기술 용어(터미널, 깃헙, 크론 등)로 후킹 금지. 결과 중심 (내 서비스 만들기, 실무 활용 등)
- 태그(🔔[오픈] 등) **없이** 카피 본문부터 바로 시작
- 공유자 이름은 카피 본문에 넣지 않음
- **bit.ly 링크 필수 포함** (카피 하단)
- 가격 표시 안 함

✅ {커리큘럼 핵심 4}

## Phase별 소구 각도
| Phase | 소구 방향 | 후킹 톤 | 하단 CTA |
|-------|---------|--------|----------|
| 오픈 | 호기심+기대감 | "요즘 ~ 안 해본 사람 없는데..." / "~ 제대로 써보고 싶었다면" | ⏰ 지금 신청 시 최대 할인가로 참여 가능! |
| 리마인드 | FOMO+사회적 증거 | "벌써 N명 돌파!!" (실제 수치) / "~ 제대로 배워보고 싶다면 이번이 딱" | ⏰ 마감 임박! 지금 신청 시 최대 할인가로 참여 가능! (마감 후 할인가 신청 불가) |
| 마감 | 긴급성+마지막 기회 | "🚨🚨 오늘 마감입니다!!" | 🚨 오늘 마감 후 최대 할인가로 신청 불가! 지금 바로 신청하세요 👇 |

### ⚠️ 후킹/리마인드 주의사항
- **고정 문구 템플릿 금지**: "벌써 100명 돌파!!" 같은 문구를 매번 복붙하면 안 됨
- 리마인드 숫자는 **실제 신청자 수** 기반 (DB 조회)
- 후킹 첫 2줄은 **그 공유회만의 핵심 소구점**에서 추출. 매번 새로 작성
- 페인포인트(😰)도 **그 주제에 맞는 실제 고민**으로 매번 다르게
- ✅ 가져갈 것도 **실제 커리큘럼에서 추출**, 복붙 금지, 재해석해서 작성

## UTM 규칙
- source: `kakaoopenchat`
- medium: `crm`
- content: `{YYMMDD}_{Phase}_{방이름}`
  - 예: `260311_오픈_냅다활용`
- **bit.ly 변환 필수** (오카방 링크 제한)
- 3개 방 각각 다른 bit.ly URL 생성 (카피는 동일, URL만 다름)

(카피 내용 30~40줄)
━━━━━━━━━━━━━━━━━━━━━
🔗 냅다활용: bit.ly/xxxxx

## 발송 방식
- Slack 승인 후 → 에밀리가 각 오카방에 수동 복붙 게시 (bit.ly만 해당 방 것으로)
