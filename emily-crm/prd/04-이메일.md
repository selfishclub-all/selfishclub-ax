# 04 — 이메일 PRD (추후 구현)

## 개요
- Stibee API 기반 이메일 발송
- **오픈 + 마감만** (리마인드 없음)
- 현재 미구현 — 전채널 중 유일하게 자동화 미완

## 발송 스케줄
| Phase | 이메일 | 대상 |
|-------|-------|------|
| Phase 2 (오픈) | 오픈 이메일 | 멤버십 전체 |
| Phase 4 (D-1) | 마감 이메일 | 미신청자 |

## Stibee API
- Endpoint: `POST /api/v1.0/auto/`
- Rate Limit: 3 req/sec
- 변수치환 지원
- 구현 시 참고: https://api.stibee.com/docs
