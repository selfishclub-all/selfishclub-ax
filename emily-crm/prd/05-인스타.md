# 05 — 인스타그램 캐러셀 PRD

## 개요
- **리마인드 Phase에 1회 게시** (Phase 3)
- 1080×1080px 캐러셀 5장
- Gemini 배경 이미지 재활용 (카플친과 동일)
- **Meta Graph API 자동 게시 구현** (이메일 외 전채널 자동화 목표)

## 캐러셀 5장 구성
| 슬라이드 | 내용 | 배경 |
|---------|------|------|
| 1. 커버 | 후킹 (관심 끌기) | Gemini 이미지 |
| 2. 문제 | 페인포인트 공감 | 다크/크림 단색 + 아이콘 |
| 3. 해결 | 솔루션 제시 | Gemini 이미지 |
| 4. 상세 | 프로그램 정보 (일정, 가격, 혜택) | 다크/크림 단색 + 텍스트 |
| 5. CTA | 행동 유도 (신청하기) | Gemini 이미지 |

## 디자인 규칙
- 사이즈: **1080×1080px** (1:1 정사각)
- Gemini 이미지: 커버/솔루션/CTA 슬라이드에 재활용
- 이미지 없는 슬라이드: 다크/크림 단색 배경 + 아이콘+텍스트
- 카플친과 동일한 Gemini 이미지 사용 가능 (비용 절약)

## 이미지 생성 파이프라인
```
1. Gemini 이미지 3개 (카플친에서 이미 생성됨 → 재활용)
2. Claude: HTML 조합 (캐러셀 카피 + 이미지 + 레이아웃) → 1080×1080 × 5장
3. Puppeteer: HTML → PNG 변환
4. Meta Graph API: 캐러셀 자동 게시
   - POST /{ig-user-id}/media (각 이미지 컨테이너 생성)
   - POST /{ig-user-id}/media (캐러셀 컨테이너 생성)
   - POST /{ig-user-id}/media_publish (게시)
   - 스케줄링: publish_time 파라미터로 예약 게시 가능
```

## 캡션 규칙
- 프로그램 핵심 가치 요약
- 해시태그 포함
- CTA: "프로필 링크에서 신청하세요"
- UTM 불필요 (인스타 링크 제한)

## 발송 방식
- Slack 전달 → 승인 → **Meta Graph API 자동 게시**
- 승인 후 dispatcher가 API 호출하여 캐러셀 자동 업로드
- 스케줄링 가능: 승인 시점에 게시 or 특정 시간 예약

## Meta Graph API 연동 (구현 필요)
- Instagram Business Account + Facebook Page 연결 필요
- 필요 권한: `instagram_basic`, `instagram_content_publish`, `pages_read_engagement`
- Access Token: Long-lived token 필요 (60일, 자동 갱신)
- 이미지: 공개 URL 필요 (SOLAPI 이미지 업로드 후 URL 활용 가능)
- Rate Limit: 25 posts/24h per account
