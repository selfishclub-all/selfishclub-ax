## 5종 구성
| # | 채널 | 위치 | 용도 | 글자수 |
|---|------|------|------|--------|
| 1 | 소식 | 카카오채널 관리자센터 > 소식 > 새 소식 작성 | 카카오 채널 피드 | ~600자 |
| 2 | 웰컴메시지 | 카카오채널 관리자센터 > 대화방 > 웰컴메시지 | 첫 진입/재진입 시 자동 메시지 | ~300자 |

## 카피 톤 규칙 (전 채널 공통)
- **자연스러운 구어체**: "~거든요", "~더라구요", "~이에요", "~해보세요"
- **AI 티 절대 금지**: "~했습니다. ~하고요." 식 setup→punchline 금지
- **비개발자 타겟일 때**: 기술 용어로 후킹 금지. 결과 중심
- **고정 문구 템플릿 금지**: 매번 공유회 주제/타겟에 맞게 새로 작성
- 가격 표시 안 함
- 이모지 적극 활용 (😰🔥👀🚨🤔 등)

## 채널별 상세 스펙

### 1. 소식 (~600자)

카카오 채널 "소식" 탭에 게시되는 콘텐츠.

```
구성:
📢 새로운 이기적공유회가 오픈되었습니다!

{프로그램명} {이모지}

📅 공유회 일정 : {일시}
⏰ 신청기간 : {마감일} (마감 후 최대 할인가로 신청 불가)

😰 {타겟 페인포인트 1}
😰 {타겟 페인포인트 2}
😰 {타겟 페인포인트 3}

{프로그램 가치 한줄 + 후킹}🔥

👇이번 공유회에서 가져가실 수 있는 것👇

✅ {핵심 가치 1}
✅ {핵심 가치 2}
✅ {핵심 가치 3}

🎁 신청만 해도 공유회 다시보기 VOD 제공
🎁 라이브 이벤트 참여 시, {혜택 요약}
```

**톤**: 상세페이지(PDP)와 톤 일치하되 카카오 소식 포맷에 맞게 재구성. **PDP 복붙 절대 금지.** 같은 정보를 카카오 소식 맥락에 맞게 재작성.

### 2. 웰컴메시지 (~300자)

카카오 채널 첫 방문자에게 보이는 메시지.

```
구성:
{이모지} 이기적공유회 <{프로그램명}>

{후킹 질문 or 공감 2줄} — 자연스러운 구어체로

{프로그램 가치 2~3줄}

📅 {일시}
```

**톤**: 따뜻하게 시작하면서 핵심 정보는 빠짐없이. 후킹 질문은 해당 공유회 주제에 맞게 매번 새로.

### 3. 채팅방메뉴 (~400자)

오픈채팅방 상단 고정 메뉴.

```
구성:
{이모지} 이기적공유회 <{프로그램명}>

{핵심 가치 1줄}

📅 {일시}
⏰ 신청마감: {마감일}

✅ {커리큘럼 핵심 1}
✅ {커리큘럼 핵심 2}
✅ {커리큘럼 핵심 3}
✅ {커리큘럼 핵심 4}

🎁 {혜택 요약 1~2줄}
```

**톤**: 간결, 스캔 가능, 정보 밀도 높게.

### 4. 인스타그램 소개 (40~60자, 1줄)

```
형식: 이모지 + [{날짜}] + 핵심 한 줄
예시: 🔥 [4/13] 코딩 없이 내 서비스 만드는 법 — 클로드코드앱 & 깃헙 실전 정복기
```

### 5. 리틀리 대표문구 (2줄)

리틀리(링크트리) 상단에 표시되는 대표 문구.

```
형식: 2줄
1줄: 이모지 + [{날짜}] + 프로그램명 + 핵심 후킹
2줄: 혜택 요약
```

**톤**: 후킹 한줄이 전부. 가장 펀치력 있는 한마디. 일시/가격 없음, 순수 후킹만.

## ⚠️ 공유회마다 달라지는 것
- **후킹 질문/페인포인트**: 주제·타겟에 맞게 매번 새로 작성
- **✅ 가져갈 것**: 실제 커리큘럼에서 추출. 복붙 금지, 재해석
- **🎁 혜택**: 상세페이지에서 크롤링. 공유회마다 라이브혜택 다름
- **📢 오프닝**: "새로운 이기적공유회가 오픈되었습니다!" 고정 가능하나, 나머지는 매번 새로

## 카피 레퍼런스 (crm-copywriter 스킬에 상세 보관)
- 소식: 4개 (영상/클프스/클코/블로그)
- 웰컴메시지: 4개 (챌린지/클프스/클코/블로그)
- 대화방메뉴: 1개 (영상)
- 리틀리: 4개 (클코/챌린지/블로그/무료공유회)

→ 레퍼런스 톤/길이를 참고하되, 카피 자체는 매번 새로 작성
