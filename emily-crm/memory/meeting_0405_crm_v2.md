## 4/5 미팅 핵심 결정사항

### 카플친 발송 현황 분석
- 멤버십 전체: 6,829명 / 전화번호 보유: 6,816명 (99.8%)
- 카플친 구독자: 2,865명 (멤버십의 42%)
- SOLAPI 친구톡 = 전화번호 기반 → 전화번호 없는 카플친에게 안 감
- 카카오모먼트 API = 카카오 내부 시스템 → 카플친 전체 발송 가능, 건당 15원 (CPMS)

**Why:** SOLAPI는 외부 서비스라 전화번호로 매칭하지만, 카카오모먼트는 카카오 내부 시스템이라 채널친구 ID로 직접 발송 가능
**How to apply:** 카플친 발송을 SOLAPI → 카카오모먼트 API로 전환

### ① 오픈알림 발송 타이밍 확정
- 크론 09:00 → 모달 → 에밀리 발송 버튼 = 발송
- 신청오픈 전에 item DB의 i_submitdate_first에 등록됨

### 카카오모먼트 API 스펙 (조사 완료)
- Host: apis.moment.kakao.com
- 인증: 비즈니스 토큰 + adAccountId
- 플로우: 캠페인 → 광고그룹 → 소재 → 발송
- 과금: CPMS 건당 15원 (타겟팅 시 20원)
- 타겟: 카플친 전체 = ageType/genderType/locationType: ALL
- Rate limit: 5초당 1회
- 사전준비: 카카오 디벨로퍼스 앱 + 비즈앱 전환 + 모먼트 API 권한 신청 + 광고계정
