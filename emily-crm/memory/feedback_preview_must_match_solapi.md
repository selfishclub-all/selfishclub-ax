슬랙 미리보기(크론 미리보기 + 수정 미리보기)는 **실제 SOLAPI 알림톡 템플릿과 동일한 텍스트**여야 한다. "참고용"으로 다른 텍스트를 넣으면 안 됨.

**Why:** 4/16 세션에서 `Code: 수정 미리보기` case 3(D-1)에 라이브혜택이 포함된 다른 텍스트가 나옴 → 에밀리가 실제 발송 내용과 다른 걸 보고 혼란. 사용자 피드백: "수정된 내용과 보낼 템플릿 그대로 와야지"

**How to apply:**
1. 새 템플릿 추가/수정 시 **SOLAPI 템플릿 원문**(prd/01-알림톡.md)을 복사해서 변수만 치환
2. `Code: 슬랙 메시지`와 `Code: 수정 미리보기` **두 곳 모두** 동시 업데이트
3. 변수 치환 패턴: `#{이름}` → `{이름}`, `#{공유회명}` → `${d.name}` 등
4. 줄바꿈, 이모지, 특수문자 위치까지 정확히 일치시킬 것
