# Timeline Generator — CRM 타임라인 자동 생성

## 입력
- data-enricher.md의 출력 (enriched data JSON)
- 필수: 모집시작일, 모집마감일, i_eventdate, i_full_schedule

## 타임라인 생성 로직

### 1. 기본 날짜 계산

```
모집기간 = 모집마감일 - 모집시작일 (일수)
D-day = i_eventdate (공유회 당일)
D-1 = i_eventdate - 1일
```

### 2. 리마인드 타이밍 자동 계산

모집기간에 비례하여 리마인드 시점을 자동 결정:

| 모집기간 | 리마인드 시점 | 설명 |
|---------|-------------|------|
| 14일 이상 | 모집시작일 + 3~4일 | 충분한 간격 |
| 7~13일 | 모집시작일 + 2~3일 | 중간 간격 |
| 4~6일 | 모집시작일 + 1~2일 | 짧은 간격 |
| 3일 이하 | 리마인드 없음 | 기간 너무 짧음 |

**계산 공식**: `리마인드일 = 모집시작일 + floor(모집기간 × 0.25)`
- 최소: 모집시작일 + 1일
- 최대: 모집마감일 - 2일 (D-1과 겹치지 않게)

### 3. Flow1: 모집 → 마감 타임라인

| Phase | 날짜 | 시간 | 채널 (R1) | 대상 |
|-------|------|------|----------|------|
| **사전세팅** | 모집시작일 전 | - | 온드미디어 4종 세팅, 카플친 신청자그룹 생성 | - |
| **오픈** | 모집시작일 | 오전 | 카플친 와이드형, 오픈채팅×3 | 카플친: 신청자제외+수신거부제외, 오픈채팅: 전체 |
| **리마인드** | 자동계산일 | 오전 | 카플친 와이드형, 오픈채팅×3, 인스타 캐러셀 | 카플친: 신청자제외, 오픈채팅: 전체 |
| **D-1** | 마감일-1 | 13:00 | 카플친 와이드형, 오픈채팅×3 | 카플친: 신청자제외, 오픈채팅: 전체 |

알림톡 (R2 추가 시):

| Phase | 날짜 | 시간 | 알림톡 | 대상 |
|-------|------|------|-------|------|
| 오픈 | 모집시작일 | 오전 | ① 오픈알림 | 멤버십 전체 |
| 리마인드 | 자동계산일 | 오전 | ② 오픈리마인드 | 미신청자 |
| D-1 | 마감일-1 | 13:00 | ③ D-1 리마인드 | 신청자 |

이메일 (R3 추가 시):

| Phase | 날짜 | 이메일 | 대상 |
|-------|------|-------|------|
| 오픈 | 모집시작일 | 오픈 안내 | 멤버십 전체 |
| D-1 | 마감일-1 | 마감 안내 | 미신청자 |

### 4. Flow2: D-day + 후속 타임라인

| Phase | 날짜 | 시간 | 채널 | 대상 |
|-------|------|------|------|------|
| **D-day 낮** | i_eventdate | 13:00 | 알림톡 ④ 당일리마인드 `[R2]` | 신청자 |
| **D-day 낮** | i_eventdate | 13:00 | 알림톡 ⑤ 할인쿠폰 `[R2]` | 미신청자 |
| **D-day 오후** | i_eventdate | 14:00 | 알림톡 ⑥-1 입장링크 `[R2]` | 신청자 |
| **D-day 저녁** | i_eventdate | 19:00 | 알림톡 ⑥-2 입장링크 `[R2]` | 신청자 |
| **공유회 시작** | i_eventdate | {i_full_schedule} | 알림톡 ⑥-3 시작됐습니다 `[R2]` | 신청자 |
| **당일 밤** | i_eventdate | 22:00 | 알림톡 ⑦ 노션발송 `[R2]` | 신청자 |
| **Post-event** | D+3~7 | - | 카플친 혜택 `[R1]`, 알림톡+이메일 VOD `[R2][R3]` | 신청자 |

### 5. 출력 포맷

```json
{
  "timeline": {
    "recruitment_period_days": 14,
    "remind_date": "2026-04-04",
    "phases": [
      {
        "phase": "사전세팅",
        "date": "2026-03-31",
        "actions": [
          {"channel": "온드미디어", "type": "리틀리", "target": "-"},
          {"channel": "온드미디어", "type": "대화방소식", "target": "-"},
          {"channel": "온드미디어", "type": "웰컴메시지", "target": "-"},
          {"channel": "온드미디어", "type": "카카오온드", "target": "-"},
          {"channel": "카플친", "type": "신청자그룹생성", "target": "-"}
        ]
      },
      {
        "phase": "오픈",
        "date": "2026-04-01",
        "actions": [
          {"channel": "카플친", "type": "와이드형", "target": "신청자제외+수신거부제외", "round": "R1"},
          {"channel": "오픈채팅", "type": "공지", "target": "전체", "url_type": "bitly", "round": "R1"},
          {"channel": "알림톡", "type": "①오픈알림", "target": "멤버십전체", "round": "R2"},
          {"channel": "이메일", "type": "오픈안내", "target": "멤버십전체", "round": "R3"}
        ]
      }
    ]
  }
}
```

## 공통 규칙 (모든 Phase 적용)

1. **UTM 자동 부착**: 모든 URL에 source/medium/campaign/content
2. **bit.ly 변환**: 오픈채팅용 URL만
3. **카플친**: 매 Phase 다른 디자인+카피
4. **오픈채팅**: 3개 동일 카피

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 모집기간 0일 (당일 오픈+마감) | 오픈만 실행, 리마인드/D-1 스킵 |
| 모집기간 1~2일 | 오픈+D-1만, 리마인드 스킵 |
| 리마인드일 = D-1 | 리마인드 하루 앞당김 |
| i_full_schedule 없음 | 기본값 20:00, 사용자에게 확인 |
| 쿠폰 정보 없음 | ⑤ 할인쿠폰 알림톡 스킵 |
