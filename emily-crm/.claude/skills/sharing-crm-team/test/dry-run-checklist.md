# Dry-Run 체크리스트: 공유회 CRM 에이전트 팀

> 실제 공유회 URL로 전체 파이프라인을 실행하여 검증

## 사전 조건

- [ ] 실제 공유회 URL 준비 (`selfishclub.xyz/sharing/{slug}`)
- [ ] Supabase 접근 가능 (SELECT only)
- [ ] Slack CRM 채널 접근 가능
- [ ] bit.ly API 접근 가능

---

## Round 1 체크리스트 (카플친 + 오픈채팅 + 인스타 + 온드미디어)

### Phase 0: 데이터 수집

- [ ] URL에서 slug 정상 추출
- [ ] Supabase `item` 테이블에서 iid, i_eventdate, i_full_schedule 조회 성공
- [ ] Webflow 상세페이지 크롤링 성공 (소개, 커리큘럼, 혜택 추출)
- [ ] 기획서.md 참조 시도 (없으면 스킵 정상)
- [ ] 멤버십 회원수, 신청자수, 미신청자수 쿼리 성공
- [ ] 누락 정보 사용자에게 질문 (모집기간 등)

### Phase 1: 타임라인

- [ ] 모집기간 기반 D-day 정상 계산
- [ ] 중간 리마인드 타이밍 자동 계산 (기간에 비례)
- [ ] Flow1(오픈→마감) 타임라인 출력
- [ ] Flow2(후속) 타임라인 출력
- [ ] director 검수 통과
- [ ] 사용자 확인 완료

### Phase 2: 카피 생성

**카플친 와이드형**
- [ ] 오픈 카피 + 디자인 스펙 생성
- [ ] 리마인드 카피 + 디자인 스펙 생성 (다른 디자인)
- [ ] 마감 카피 + 디자인 스펙 생성 (마감 디자인)
- [ ] 혜택 카피 생성 (Post-event)

**오픈채팅 3개**
- [ ] 오픈 공지 카피 생성
- [ ] 리마인드 카피 생성 (다른 카피)
- [ ] 마감 공지 카피 생성 (마감 카피)

**인스타 캐러셀**
- [ ] 슬라이드 텍스트 + 캡션 + 해시태그 생성 (리마인드 Phase만)

**온드미디어 4종**
- [ ] 리틀리 카피 생성
- [ ] 대화방소식 카피 생성
- [ ] 웰컴메시지 카피 생성
- [ ] 소식 카피 생성

### UTM + bit.ly

- [ ] 모든 URL에 UTM 파라미터 부착 (source/medium/campaign/content)
- [ ] 오픈채팅용 URL만 bit.ly 변환
- [ ] UTM source가 채널별로 올바른지 (kakao/openchat/instagram 등)

### 검수

- [ ] team-lead 검수 통과 (카피 완전성, 변수 정확성, URL, 톤)
- [ ] director 검수 통과 (전체 정합성)

### Phase 3: Slack 프레젠테이션

- [ ] CRM 채널에 타임라인 순서대로 메시지 표시
- [ ] Phase별 메인 메시지 + 채널별 스레드 답글
- [ ] 카피가 코드블록으로 복사 가능하게 표시
- [ ] 사용자 승인/수정 피드백 수신

### 아티팩트

- [ ] `.omc/crm-artifacts/{slug}/` 디렉토리 생성
- [ ] manifest.json에 전체 산출물 참조 저장
